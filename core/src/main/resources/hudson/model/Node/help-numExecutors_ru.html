Эта опция контролирует количество одновременных сборок, которые может производить Jenkins.

Для начала стоит установить это значение равным количеству процессоров в вашей системе. Увеличивая это значение далее, вы можете заметить увеличение продолжительности каждой сборки, однако результирующая пропускная способность может увеличиться. Например, одна сборка может быть ограничена производительностью процессора, а другая, работающая в то же время - вводом-выводом. Тогда вторая сборка может в этот момент воспользоваться лишней пропускной способностью ввода-вывода

Агенты (узлы кроме встроенного) должны иметь минимум один сборщик. Чтобы приостановить выполнение сборок на узле, используйте кнопку Пометить данный узел как временно недоступный на странице настройки узла.

Для встроенного узла, установите значение в 0 - это запретит ему выполнять сборки локально на контроллере. Примечание: Встроенный узел всегда может выполнять легковесные задачи, включая задачу верхнего уровня Pipeline.
